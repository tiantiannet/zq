意见反馈

页面说明：

  1. 导航栏右侧显示当前登录的个人用户姓名，点击后显示下拉菜单，可以进行个人设置或者注销操作
  2. 个人端登陆成功后回跳转到该页面，每次默认显示八个挑战职位
  3. 可以通过下拉框和搜索栏对职位进行搜索
  4. 当页面拉到最下方时，如果还有其他职位的挑战则再加载八个
  5. 点击“接受挑战”按钮，则会跳转到该挑战页面
  6. 点击排序右侧的按钮，可以分别按照“时间”“热度”“薪资”进行排序
